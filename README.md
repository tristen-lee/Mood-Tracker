Mood-Tracker
Built With Python 3.14

A Python command-line mood tracking app built to help individuals with Bipolar Disorder log daily mood, sleep, and mental health data. Track patterns over time and receive supportive feedback to help you reflect on your well-being.

Features
Daily logging of sleep hours, mood score (1-10), mania, psychosis, depression, suicidal thoughts, and notes
Yes/No inputs for quick, simple data entry
Average mood calculation with friendly interpretations
Supportive suggestions if you choose to receive tips
Data storage using JSON for persistent local tracking
CLI menu for logging entries, viewing data, and analytics
JSON for local data storage


Getting Started

Clone this repository:
git clone https://github.com/tristen-lee/Mood-Tracker.git

Navigate to the project folder:
cd Mood-Tracker

Run the tracker:
python -m core.tracker


Enter your daily sleep, mood, and mental health status
Choose whether you want tips when prompted
View your logged data or calculate average mood
Receive friendly interpretations to help reflect on patterns

Example Interpretations:

Low average mood → gentle suggestions for self-care
Balanced mood → reassurance and encouragement
High mood → celebrate positive streaks while tracking stability
